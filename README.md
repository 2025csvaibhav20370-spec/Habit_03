# Smart Productivity Tracker

A modern productivity web app built with React, Vite, and Tailwind CSS.

## Features
- Login / Register (localStorage based)
- Dashboard with charts
- Task Manager (categories, due dates, priorities)
- Habit Tracker (streaks, custom habits)
- Coding Challenge tracker with timer, heatmap, and analytics

## How to Run Locally

### Requirements
- Node.js 18 or higher (download from https://nodejs.org)
- npm (comes with Node.js)

### Steps

1. Open a terminal and go into this folder:
   ```
   cd smart-productivity-tracker
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Open your browser and go to:
   ```
   http://localhost:5173
   ```

That's it! All your data is saved in your browser automatically.
